# EvHDR-NeRF: Building High Dynamic Range Radiance Fields with Single Exposure Images and Events

**EvHDR-NeRF** builds a **high dynamic range (HDR) radiance field** from a **single exposure images** and **event streams**. This work was accepted at **AAAI 2025**.

- **Project page:** [EvHDR-NeRF website](https://zehaoc.github.io/EvHDR-NeRF/)
- **Code:** In the [`code/`](code/) directory.
